Gateway Resilience (Circuit Breaker)

How the System handles a 503 Service Unavailable from Python

Circuit Open
Upstream Error HTTP 503

// Python Service Offline

ConnectException: Connection refused

Gateway attempt to reach port 8000 failed.

Graceful Fallback Status: 200 OK 
{
  "status": "degraded",
  "inferenceService": "down",
  "timestamp": 1774323746533
}

The Gateway intercepts the error and returns a safe "Degraded" DTO.
